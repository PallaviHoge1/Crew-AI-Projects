# Pandas Data Visualization

**Difficulty:** beginner

**Estimated hours:** 10

## Description

Create interactive plots and charts using pandas data frames.

## Steps

1. Install necessary libraries: matplotlib, seaborn
2. Load a sample dataset (e.g., tip dataset)
3. Visualize the dataset using histograms, boxplots, scatter plots

## Required Skills

- pandas
- matplotlib
- seaborn

## How to run

1. Create virtualenv: `python -m venv .venv`
2. Activate and install `pip install -r requirements.txt`
3. Run `python main.py` or open `notebook.ipynb` in Jupyter.
