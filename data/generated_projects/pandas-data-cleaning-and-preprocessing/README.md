# Pandas Data Cleaning and Preprocessing

**Difficulty:** intermediate

**Estimated hours:** 20

## Description

Clean and preprocess a pandas data frame for analysis.

## Steps

1. Handle missing values using drop, fill, or impute methods
2. Remove outliers using visualizations or statistical methods

## Required Skills

- pandas
- numpy
- sklearn

## How to run

1. Create virtualenv: `python -m venv .venv`
2. Activate and install `pip install -r requirements.txt`
3. Run `python main.py` or open `notebook.ipynb` in Jupyter.
