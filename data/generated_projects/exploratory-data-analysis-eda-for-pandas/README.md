# Exploratory Data Analysis (EDA) for Pandas

**Difficulty:** beginner

**Estimated hours:** 15

## Description

Perform EDA on a pandas data frame to understand the distribution of variables.

## Steps

1. Load a sample dataset (e.g., iris dataset)
2. View summary statistics and data types

## Required Skills

- pandas
- numpy

## How to run

1. Create virtualenv: `python -m venv .venv`
2. Activate and install `pip install -r requirements.txt`
3. Run `python main.py` or open `notebook.ipynb` in Jupyter.
